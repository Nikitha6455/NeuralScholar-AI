# Few-Shot Prompting Strategy

## Purpose

NeuralScholar AI uses Few-Shot Prompting to improve answer quality and consistency.

## Example 1

Question:
What is RAG?

Answer:
RAG stands for Retrieval Augmented Generation. It combines information retrieval with language generation.

## Example 2

Question:
What is FAISS?

Answer:
FAISS is a vector database used for efficient similarity search.
